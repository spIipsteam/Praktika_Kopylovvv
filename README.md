Описание

«AvtoObuvka» — это веб-приложение для подбора и покупки автомобильных шин. Сайт предлагает удобный интерфейс для фильтрации шин по сезону, размеру и бренду, добавления товаров в корзину и оформления заказа. Разработан как учебный проект с использованием HTML, CSS и JavaScript.

Установка и запуск

1. Скачайте файлы проекта:
   - `index.html`
   - (При необходимости) дополнительные CSS/JS файлы, если проект будет разделён

2. Откройте проект:
   - Запустите `index.html` в любом современном браузере (Chrome, Firefox, Edge)
   - Или используйте локальный сервер для разработки (например, Live Server в VS Code)

3. Использование:
   - Приложение работает полностью на клиентской стороне
   - Все данные (товары, корзина) хранятся в localStorage браузера

Использование

AvtoObuvka — это удобный онлайн-магазин автомобильных шин с функциями:
- Фильтрация товаров по сезону, ширине и бренду
- Интерактивная корзина с добавлением/удалением товаров
- Адаптивный дизайн для мобильных устройств
- Визуальный каталог с изображениями товаров
